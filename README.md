# kursayin_nakhagits
Հարթակը հնարավորություն է տալիս կայքից օգտվող ուսանողներին մեկ տեղում տեսնել
իրենց ակադեմիական վիճակագրությունը, հետևել ինտերակտիվ դասացուցակին,
արագ հաղորդակցվել կուրսեցիների ու դասախոսների հետ ներկառուցված չատերի միջոցով,
ինչպես նաև առանձին առարկայական էջերում ներբեռնել անհրաժեշտ PDF դասագրքերն ու դիտել վիդեոդասերը։

Հետագայում նախատեսվում է ավելացնել թե՛ վեբ կայքի ֆունկցիոնալ բաժինները,
թե՛ դրա դինամիկությունը JavaScript-ի միջոցով։ 
Աշխատանքը կատարելագործելու և օգտատերերի տվյալներն ու ուսումնական նյութերը
պահպանելու նպատակով նախատեսված է նաև Տվյալների բազաների (SQL) ամբողջական ինտեգրում։
